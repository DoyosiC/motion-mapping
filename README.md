# motion-mapping

# 手検出によるゲーム操作システム

カメラ映像から手の動きを検出し、非接触でゲーム操作を行うリアルタイムインタラクションシステムである。  
MediaPipeとOpenCVを用いて手の位置を認識し、Pygame上のキャラクター制御へと変換する。

---

## 概要

本システムは以下の要素で構成される。

- 手のランドマーク検出（MediaPipe Hands）
- 映像処理（OpenCV）
- 入力変換（手の位置から操作コマンドへの変換）
- ゲーム処理（Pygame）

カメラ映像から取得した手の位置（上下）および左右の手の識別を用いて、直感的な操作を実現する。

## 操作方法

手の位置に応じて以下の操作を行う。

| 手 | 位置 | 動作 |
|----|------|------|
| 左手 | 上 | ジャンプ |
| 右手 | 上 | 右移動 |
| 右手 | 下 | 左移動 |

画面を縦方向に3分割し、「上・中・下」で判定する。

## システム構成

### 処理フロー

1. カメラから映像を取得  
2. BGRからRGBへ変換  
3. MediaPipeによる手のランドマーク検出  
4. 指先座標から手の位置（上・中・下）を判定  
5. ゲーム入力へ変換  
6. Pygameで描画および更新  

---

### ゲームロジック概要

- 重力を考慮したジャンプ処理
- 敵キャラクタのランダム生成
- 当たり判定によるゲーム終了
if player.rect.colliderect(enemy.rect):
print("ゲームオーバー")

- カメラ番号は環境に応じて変更が必要
